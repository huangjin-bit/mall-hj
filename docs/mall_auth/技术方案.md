# mall-auth 鉴权服务 - 技术方案

## 技术栈
- Spring Boot 3.5.14 + Spring Security（无状态模式）
- JJWT 0.12.6（HS256签名）
- Redis（Token缓存 + 短信验证码）
- MyBatis-Plus 3.5.7（读取 mall_member 库）
- Spring Cloud OpenFeign（供其他服务调用）

## 包结构
com.hj.mall.auth/
├── config/          # SecurityConfig, JwtConfig, MybatisPlusConfig
├── controller/      # AuthController
├── dto/             # LoginDTO, RegisterDTO, TokenDTO, SmsCodeDTO
├── entity/          # Member, MemberAuth, MemberLoginLog
├── feign/           # AuthFeignClient, AuthFeignFallback, AuthFeignController
├── mapper/          # MemberMapper, MemberAuthMapper, MemberLoginLogMapper
├── service/         # AuthService, TokenService
└── util/            # JwtUtil

## JWT 方案
- 算法：HMAC-SHA256
- Secret：配置文件中 jwt.secret
- 有效期：24h（86400000ms）
- Payload：subject=username, claims={memberId}
- 存储：Redis key=auth:token:{memberId}, value=token, TTL=24h

## Spring Security 配置
- 无状态（SessionCreationPolicy.STATELESS）
- 放行路径：/auth/**, /feign/**
- 禁用：CSRF, httpBasic, formLogin

| 路径 | 方法 | 说明 |
|---|---|---|
| /feign/auth/validate | GET | 验证Token有效性 |
