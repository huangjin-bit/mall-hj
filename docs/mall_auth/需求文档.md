# mall-auth 鉴权服务 - 需求文档

## 服务概述
鉴权服务，负责用户登录、注册、JWT Token 管理和短信验证码发送。

## 端口
8081

## 数据库
共享 mall_member 数据库

## 功能需求

### F1 用户名密码登录
- 输入：用户名 + 密码
- 验证：查询 ums_member_auth 表（identity_type=USERNAME），BCrypt 校验密码
- 输出：JWT Token（24h有效）
- 登录成功记录 ums_member_login_log

### F2 手机验证码登录
- 输入：手机号 + 短信验证码
- 验证：从 Redis 获取验证码比对（key: auth:sms:{phone}，5min过期）
- 输出：JWT Token

### F3 用户注册
- 输入：用户名 + 手机号 + 密码
- 创建 ums_member 记录
- 创建 ums_member_auth 记录（USERNAME + PHONE 两种身份类型）
- 密码使用 BCrypt 加密

### F4 发送短信验证码
- 输入：手机号
- 生成6位随机验证码
- 存入 Redis（5min过期）
- 限流：同一手机号60秒内只能发送一次

### F5 Token 验证
- 验证 JWT 签名和有效期
- 验证 Redis 中是否存在（支持主动注销）
