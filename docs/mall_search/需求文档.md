# mall-search 商品搜索服务 - 需求文档

## 服务概述
基于 Elasticsearch 的商品搜索服务。

## 端口
8087

## 功能需求

### F1 商品搜索
- 关键词搜索（IK分词）
- 分类过滤
- 品牌过滤
- 价格区间过滤
- 排序（价格升/降序、最新）
- 分页

### F2 商品上架（同步ES）
- 商品发布时，将 SKU 信息同步到 Elasticsearch
- 支持批量上架

### F3 商品下架
- 从 Elasticsearch 删除 SKU 索引
- 支持批量下架

## 搜索字段
- skuName, spuName, spuDescription（IK分词）
- categoryId, brandId（精确过滤）
- price（范围过滤）
