# mall-search 商品搜索服务 - 技术方案

## 技术栈
- Spring Boot 3.5.14
- Spring Data Elasticsearch
- Elasticsearch 8.x（需安装 IK Analysis 插件）

## 包结构
com.hj.mall.search/
├── controller/      # SearchController, ElasticSaveController
├── dto/             # SearchParam
├── feign/           # SearchFeignClient, SearchFeignFallback, SearchFeignController
├── model/           # SkuEsModel（ES文档模型）
├── repository/      # SkuEsRepository
├── service/         # SearchService, SearchServiceImpl
└── vo/              # SearchResult

## ES 索引设计
索引名：mall_product
| 字段 | 类型 | 分词器 | 说明 |
|---|---|---|---|
| skuId | Long | - | 主键 |
| spuId | Long | - | SPU ID |
| skuName | Text | ik_max_word / ik_smart | SKU名称 |
| spuName | Text | ik_max_word / ik_smart | SPU名称 |
| spuDescription | Text | ik_max_word | SPU描述 |
| price | Double | - | 售价 |
| categoryId | Long | - | 分类ID |
| brandId | Long | - | 品牌ID |
| publishStatus | Integer | - | 上架状态 |

## 查询构建
使用 ElasticsearchOperations + BoolQueryBuilder 构建动态查询：
- must: multiMatch(skuName, spuName, spuDescription)
- filter: term(categoryId), term(brandId), range(price)
- sort: field(price/createTime) + order(ASC/DESC)
