# mall_member 会员库 — 需求文档

## 1. 模块概述

`mall_member` 是 Mall-HJ 项目的会员与认证中心数据库，服务于 `mall-auth`（认证服务）和 `mall-member`（会员服务）两个微服务。

核心职责：
- 会员注册、登录、个人信息管理
- 多方式认证绑定（账密 / 手机+验证码 / 微信 / 支付宝）
- 作为独立 OAuth2 认证中心，支持未来其他项目对接
- 会员等级、积分、成长值体系
- 用户画像与标签（为推荐系统提供数据基础）
- 收货地址管理

---

## 2. 功能需求

### 2.1 会员管理

| 功能 | 描述 |
|---|---|
| 注册 | 支持用户名+密码、手机号+验证码注册 |
| 个人信息 | 修改昵称、头像、性别、生日等 |
| 用户画像 | 年龄段、职业、价格敏感度、购物偏好、偏好分类 |
| 账号状态 | 管理员可禁用/启用账号 |

### 2.1.1 用户画像与标签

会员主表内置画像字段，支持推荐系统：

| 字段 | 用途 |
|---|---|
| age_group | 年龄段（18以下/18-25/26-35/36-45/45以上） |
| occupation | 职业（学生/白领/自由职业/其他） |
| price_sensitivity | 价格敏感度（1低/2中/3高） |
| favorite_categories | 偏好分类ID（逗号分隔） |
| shopping_preference | 购物偏好（品质优先/性价比/品牌导向） |

标签体系通过独立标签表+关联表实现：

- 标签类型：人口属性(DEMOGRAPHIC)、行为特征(BEHAVIOR)、消费偏好(PREFERENCE)、自定义(CUSTOM)
- 打标方式：系统自动(AUTO) 或 人工标记(MANUAL)
- 示例标签：高频用户、价格敏感、母婴人群、数码爱好者、品牌忠诚等

| 功能 | 描述 |
|---|---|
| 注册 | 支持用户名+密码、手机号+验证码注册 |
| 个人信息 | 修改昵称、头像、性别、生日等 |
| 账号状态 | 管理员可禁用/启用账号 |

### 2.2 多方式登录

| 登录方式 | 说明 |
|---|---|
| 用户名+密码 | BCrypt 加密存储，密码强度校验 |
| 手机号+验证码 | Redis 存验证码，5分钟过期，1分钟限流 |
| 微信扫码登录 | OAuth2 获取 openId 绑定 |
| 支付宝扫码登录 | OAuth2 获取 userId 绑定 |

一个会员可绑定多种登录方式，在个人中心管理绑定/解绑。

### 2.3 OAuth2 认证中心

mall-auth 作为独立认证中心，采用**授权码模式 (Authorization Code)**。

- 支持注册多个客户端应用（商城PC端、商城APP、未来项目等）
- Token 策略：Redis + JWT（支持主动失效）
- 支持刷新令牌 (Refresh Token)

### 2.4 会员等级体系

| 等级 | 所需成长值 | 权益 |
|---|---|---|
| 普通会员 | 0 | 基础功能 |
| 银卡会员 | 1,000 | 满99免运费、评价双倍积分、98折 |
| 金卡会员 | 5,000 | 满59免运费、评价3倍积分、95折 |
| 铂金会员 | 15,000 | 全场免运费、评价4倍积分、9折 |
| 钻石会员 | 50,000 | 全场免运费、评价6倍积分、85折 |

### 2.5 积分体系

- 获取途径：下单、评价、签到、管理员手动调整
- 使用途径：下单抵扣、积分兑换
- 所有变动记录流水，包含变动前后余额

### 2.6 成长值体系

- 获取途径：下单、评价、签到
- 成长值累计达到阈值自动升级
- 所有变动记录流水

### 2.7 收货地址

- 一个会员可维护多个收货地址
- 支持设置默认地址（唯一）
- 包含省/市/区/详细地址

### 2.8 登录日志

- 记录每次登录的时间、IP、城市、设备信息
- 区分登录成功/失败
- 支持安全审计和异常检测

## 3. 数据库表清单

| # | 表名 | 类型 | 说明 |
|---|---|---|---|
| 1 | ums_member | 业务表 | 会员主表（含画像字段） |
| 2 | ums_member_auth | 业务表 | 认证绑定（账密/手机/微信/支付宝） |
| 3 | ums_member_level | 配置表 | 会员等级 |
| 4 | ums_member_receive_address | 业务表 | 收货地址 |
| 5 | ums_member_tag | 配置表 | 用户标签定义 |
| 6 | ums_member_tag_relation | 业务表 | 用户-标签关联 |
| 7 | ums_member_login_log | 日志表 | 登录日志 |
| 8 | ums_integration_record | 日志表 | 积分流水 |
| 9 | ums_growth_record | 日志表 | 成长值流水 |
| 10 | sys_client | 配置表 | OAuth2 客户端注册 |
| 11 | sys_oauth2_authorization | 日志表 | OAuth2 授权记录 |
| 12 | sys_oauth2_consent | 业务表 | OAuth2 用户授权同意 |
