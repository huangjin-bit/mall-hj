# mall-gateway API 网关 - 技术方案

## 技术栈
- Spring Cloud Gateway（WebFlux）
- JJWT 0.12.6（JWT本地验证）
- Spring Cloud LoadBalancer
- Alibaba Sentinel（限流，待集成）

## 包结构
com.hj.mall.gateway/
├── config/       # JwtConfig
├── filter/       # AuthGlobalFilter
└── util/         # JwtUtil

## 鉴权流程
1. 请求进入网关
2. AuthGlobalFilter（order=-1）检查路径是否在白名单
3. 白名单路径直接放行
4. 非白名单路径提取 Authorization: Bearer {token}
5. JwtUtil 验证 JWT 签名+有效期
6. 验证通过：注入 X-User-Id, X-Username 头，转发到下游
7. 验证失败：返回 401 Unauthorized

## 下游服务验证
下游服务通过 mall-common 的 UserContextFilter 读取 X-User-Id/X-Username 头。
