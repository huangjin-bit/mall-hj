# mall-gateway API 网关 - 需求文档

## 服务概述
API网关，负责路由转发、JWT鉴权、CORS跨域。

## 端口
8080

## 功能需求

### F1 路由转发
所有 /api/{service}/** 请求转发到对应微服务，StripPrefix=1。

| 路径 | 目标服务 |
|---|---|
| /api/auth/** | mall-auth |
| /api/member/** | mall-member |
| /api/product/** | mall-product |
| /api/order/** | mall-order |
| /api/seckill/** | mall-seckill |
| /api/ware/** | mall-ware |
| /api/search/** | mall-search |
| /api/thirdparty/** | mall-third-party |
| /api/cart/** | mall-cart |

### F2 JWT 鉴权
- 从 Authorization 头提取 Bearer Token
- 本地验证 JWT 签名和有效期
- 提取 memberId/username 注入 X-User-Id/X-Username 请求头
- 白名单路径免鉴权：/auth/**, /categories/**, /brands/**, /spu/**, /sku/**, /feign/**

### F3 CORS 跨域
- 允许所有来源
- 允许所有方法和头
