# mall-cart 购物车服务 - 需求文档

## 服务概述
基于 Redis 的购物车服务，无数据库持久化。

## 端口
8089

## 功能需求

### F1 查看购物车
- 获取当前用户的所有购物车商品
- 计算总数量、总价格（勾选项）

### F2 添加商品
- 添加 SKU 到购物车
- 已存在的商品数量+1

### F3 修改数量
- 修改指定 SKU 的购买数量

### F4 勾选/取消勾选
- 单个商品勾选/取消
- 全选/全不选

### F5 删除商品
- 删除一个或多个购物车商品

### F6 获取勾选项（Feign）
- 订单服务调用，获取已勾选的购物车商品
- 下单后清除已勾选项
