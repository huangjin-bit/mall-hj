# mall-cart 购物车服务 - 技术方案

## 技术栈
- Spring Boot 3.5.14
- Spring Data Redis（StringRedisTemplate）
- Jackson（CartItem JSON序列化）
- Spring Cloud OpenFeign

## 包结构
com.hj.mall.cart/
├── controller/   # CartController
├── feign/        # CartFeignClient, Fallback, Controller
├── model/        # CartItem
├── service/      # CartService, CartServiceImpl
└── vo/           # CartVO

## Redis 存储设计
- Key: cart:{memberId}
- Type: Hash
- Field: {skuId}
- Value: CartItem JSON

## CartItem 结构
| 字段 | 说明 |
|---|---|
| skuId | SKU ID |
| spuId | SPU ID |
| skuName | SKU名称 |
| skuImg | 商品图片 |
| price | 售价 |
| originalPrice | 原价 |
| quantity | 数量 |
| isChecked | 是否勾选 |
| saleAttrs | 销售属性值 |

## 性能考量
- 全部操作在内存中完成，响应快
- 无DB依赖，服务轻量
- 适合高并发场景（秒杀加购等）
